可以。下面这套命令默认你的服务器是全新 `Ubuntu 22.04`，并且你要按这套结构部署：

- 代码：`/srv/apps/zjuaaa-site`
- 数据：`/srv/data/zjuaaa-site`
- 备份：`/srv/backups/zjuaaa-site`

把其中的：
- `YOUR_DOMAIN`
- `YOUR_SERVER_IP`
- `CHANGE_ME_STRONG_PASSWORD`
- `CHANGE_ME_NEXTAUTH_SECRET`

替换成你自己的值。

**1. 基础环境**

```bash
sudo apt update
sudo apt upgrade -y
sudo apt install -y git curl nginx unzip build-essential
```

**2. 安装 Node.js 22 和 PM2**

```bash
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt install -y nodejs
sudo npm install -g pm2
node -v
npm -v
pm2 -v
```

**3. 创建目录**

```bash
sudo mkdir -p /srv/apps
sudo mkdir -p /srv/data/zjuaaa-site/uploads
sudo mkdir -p /srv/backups/zjuaaa-site
sudo chown -R $USER:$USER /srv/apps /srv/data /srv/backups
```

**4. 拉取源码**

```bash
cd /srv/apps
git clone https://github.com/Ray8mu34/ZJUAAA_Web.git zjuaaa-site
cd /srv/apps/zjuaaa-site
```

**5. 安装依赖**

```bash
npm install
```

**6. 创建数据目录下的环境变量文件**

```bash
cat > /srv/data/zjuaaa-site/.env <<'EOF'
DATABASE_URL="file:/srv/data/zjuaaa-site/dev.db"
UPLOAD_DIR="/srv/data/zjuaaa-site/uploads"
NEXTAUTH_URL="http://YOUR_SERVER_IP"
NEXTAUTH_SECRET="CHANGE_ME_NEXTAUTH_SECRET"
ADMIN_USERNAME="admin"
ADMIN_PASSWORD="CHANGE_ME_STRONG_PASSWORD"
EOF
```

**7. 链接 `.env` 到代码目录**

```bash
ln -sf /srv/data/zjuaaa-site/.env /srv/apps/zjuaaa-site/.env
```

**8. 初始化数据库和 Prisma**

```bash
cd /srv/apps/zjuaaa-site
npx prisma db push --skip-generate
npx prisma generate
```

**9. 如需创建管理员种子账号**

如果项目里有种子脚本就执行：

```bash
node scripts/seed-admin.mjs
```

如果没有，也没关系，按你们项目当前后台逻辑来。

**10. 构建项目**

```bash
cd /srv/apps/zjuaaa-site
npm run build
```

**11. 用 PM2 启动**

```bash
cd /srv/apps/zjuaaa-site
pm2 start npm --name zjuaaa-site -- start
pm2 save
pm2 startup
```

执行 `pm2 startup` 后，终端会返回一条命令，把那条命令再执行一次。

**12. 检查服务是否启动**

```bash
pm2 status
pm2 logs zjuaaa-site --lines 50
curl http://127.0.0.1:3000
```

**13. 配置 Nginx**

```bash
sudo tee /etc/nginx/sites-available/zjuaaa-site > /dev/null <<'EOF'
server {
    listen 80;
    server_name YOUR_DOMAIN YOUR_SERVER_IP;

    client_max_body_size 64M;

    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
}
EOF
```

**14. 启用站点**

```bash
sudo ln -sf /etc/nginx/sites-available/zjuaaa-site /etc/nginx/sites-enabled/zjuaaa-site
sudo rm -f /etc/nginx/sites-enabled/default
sudo nginx -t
sudo systemctl restart nginx
sudo systemctl enable nginx
```

**15. 开放防火墙**

```bash
sudo ufw allow OpenSSH
sudo ufw allow 'Nginx Full'
sudo ufw --force enable
sudo ufw status
```

**16. 访问测试**

浏览器打开：

```text
http://YOUR_SERVER_IP
```

如果绑了域名，也可以打开：

```text
http://YOUR_DOMAIN
```

**17. 配 HTTPS（有域名时）**

```bash
sudo apt install -y certbot python3-certbot-nginx
sudo certbot --nginx -d YOUR_DOMAIN
```

**18. 后续更新命令**

以后代码更新就这几条：

```bash
cd /srv/apps/zjuaaa-site
git pull
npm install
npx prisma generate
npm run build
pm2 restart zjuaaa-site
```

**19. 备份命令**

```bash
tar -czf /srv/backups/zjuaaa-site/uploads-$(date +%F-%H%M%S).tar.gz /srv/data/zjuaaa-site/uploads
cp /srv/data/zjuaaa-site/dev.db /srv/backups/zjuaaa-site/dev-$(date +%F-%H%M%S).db
```
