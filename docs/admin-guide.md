# 后台使用指南

本文面向日常维护网站内容的同学。正常更新网站内容时，不需要修改源码，登录后台操作即可。

## 1. 后台入口

- 后台地址：`/admin`
- 登录地址：`/admin/login`
- 初始管理员账号只建议用于首次进入后台，进入后应尽快修改密码或新建个人管理员账号。

## 2. 日常维护原则

- 先上传图片，再创建或编辑内容。
- 知识科普和社团活动通常是“卡片入口 + 外部链接”模式。
- 天文手册是站内长期正文库，适合保存完整 Markdown 内容。
- 不确定图片路径时，优先使用媒体库选择，不要手动填写路径。
- 发布前检查状态是否为“已发布”。

## 3. 后台模块概览

| 模块 | 用途 |
| --- | --- |
| 首页管理 | 维护首页标题、副标题、按钮、Logo、首页展示内容等 |
| 站点设置 | 维护关于我们、加入我们、联系我们、历届成员、手册内容清单等 |
| 知识科普 | 维护科普文章卡片和公众号链接 |
| 社团活动 | 维护活动卡片、时间、地点、封面和跳转链接 |
| 天文手册 | 维护站内手册文章 |
| 手册栏目管理 | 创建和维护手册栏目 |
| 手册批量导入 | 用 ZIP 一次导入一个栏目的文章和图片 |
| 天文摄影 | 维护摄影作品和参数 |
| 媒体库 | 上传和选择图片 |
| 管理员 | 新增管理员、停用账号、重置密码 |

## 4. 媒体库与图片

推荐流程：

1. 进入后台媒体库。
2. 上传图片。
3. 回到对应内容模块。
4. 在封面图、Logo、正文插图等位置选择刚上传的图片。
5. 保存内容。

手册编辑器支持拖拽或粘贴图片，图片会自动上传到媒体库并插入正文。

图片建议：

- 知识科普和社团活动封面建议使用接近 `4:3` 的图片。
- 手册栏目封面建议使用横图。
- 天文摄影作品应尽量上传清晰原图，并填写作者和器材信息。

## 5. 知识科普

知识科普主要用于展示公众号文章入口。

常用字段：

| 字段 | 是否必填 | 说明 |
| --- | --- | --- |
| 标题 | 必填 | 卡片标题 |
| 作者 | 必填 | 作者姓名 |
| 摘要 | 可选 | 卡片简介，前台会自动截断 |
| 封面图 | 可选 | 从媒体库选择 |
| 公众号文章链接 | 可选 | 填完整链接，例如 `https://mp.weixin.qq.com/...` |
| 是否精选 | 可选 | 精选后可出现在首页推荐 |
| 状态 | 必填 | 只有已发布内容会在前台显示 |

前台展示方式：

- 卡片显示标题、摘要、作者和封面。
- 图片仅展示。
- 图片下方文字区域可点击跳转。

## 6. 社团活动

社团活动用于展示观星、讲座、路边天文、联动活动等入口。

常用字段：

| 字段 | 是否必填 | 说明 |
| --- | --- | --- |
| 标题 | 必填 | 活动标题 |
| 摘要 | 可选 | 活动简介，前台会自动截断 |
| 封面图 | 可选 | 从媒体库选择 |
| 地点 | 可选 | 活动地点 |
| 开始时间 | 可选 | 活动开始时间 |
| 结束时间 | 可选 | 活动结束时间 |
| 外部链接 | 可选 | 公众号推文、报名页或活动详情页 |
| 状态 | 必填 | 只有已发布内容会在前台显示 |

前台展示方式：

- 卡片显示标题、摘要、地点、时间和封面。
- 有外部链接时，文字区域可点击跳转。

## 7. 天文手册

天文手册用于保存长期资料，正文存储在网站数据库中。

### 7.1 栏目管理

进入后台的手册栏目管理，可以：

- 新增栏目。
- 修改栏目标题、简介、封面和排序。
- 控制栏目是否对外可见。
- 删除栏目。

字段说明：

| 字段 | 说明 |
| --- | --- |
| 栏目标题 | 前台显示名称 |
| Slug | 可留空，由系统生成；也可手动填写稳定英文路径 |
| 简介 | 栏目说明 |
| 封面图 | 栏目页或入口图 |
| 排序值 | 数字越小越靠前 |
| 对外可见 | 关闭后前台不展示 |

### 7.2 文章管理

进入后台的天文手册，可以：

- 新增文章。
- 按栏目筛选文章。
- 编辑 Markdown 正文。
- 插入图片和公式。
- 发布或保存草稿。

字段说明：

| 字段 | 说明 |
| --- | --- |
| 所属栏目 | 必选 |
| 章节标题 | 必填 |
| 排序值 | 数字越小越靠前；留空时按默认规则 |
| 作者 | 可选 |
| 简介 | 可选，用于文章列表 |
| Markdown 正文 | 必填，支持公式和图片 |
| 状态 | 发布后前台可见 |

### 7.3 批量导入

批量导入适合一次导入一个栏目的一组 Markdown 文件和图片。

推荐 ZIP 结构：

```text
栏目名称/
├── 01-文章标题.md
├── 02-另一篇文章.md
└── images/
    ├── image-1.jpg
    └── image-2.png
```

Markdown 可使用 frontmatter：

```yaml
---
title: "为什么行星会逆行？"
slug: why-retrograde
author: 张三
summary: 文章简介
---
```

导入规则：

- `title` 优先使用 frontmatter，否则从文件名提取。
- `slug` 优先使用 frontmatter，否则自动生成。
- `author`、`summary` 从 frontmatter 读取。
- 排序值优先从文件名前缀提取，例如 `01-xxx.md`。
- 图片会上传到媒体库，并自动替换 Markdown 中的引用路径。

## 8. 天文摄影

天文摄影用于展示作品和拍摄信息。

常用字段：

- 标题
- 摄影者
- 作品图片
- 作品说明
- 目标天区
- 拍摄地点
- 主镜、相机、赤道仪、滤镜、软件
- 状态

只有已发布作品会在前台显示。

## 9. 关于我们与历届成员

关于我们页面包括：

- 社团介绍
- 照片墙
- 历届成员名单

历届成员支持逐条添加或批量导入。批量导入格式为：

```text
2024,张三,会长
2024,李四,宣传部部长
2023,王五,学术部部长
```

支持逗号、中文逗号、制表符等分隔符。前台会按年份倒序显示，最新年份在前。

## 10. 管理员账号

管理员模块可用于：

- 新增管理员。
- 停用账号。
- 重置密码。

建议：

- 每位维护者使用自己的账号。
- 不要多人共用初始管理员账号。
- 密码重置后应提醒对方尽快修改。

## 11. 不需要改源码的内容

以下内容都应通过后台维护：

- 首页文案、按钮、Logo
- 页脚信息
- 关于我们、加入我们、联系我们文案
- 历届成员名单
- 知识科普卡片
- 社团活动卡片
- 天文手册栏目和文章
- 天文摄影作品
- 管理员账号

需要新增页面、大改页面布局、接入新服务或修改数据库结构时，再联系开发维护者。
