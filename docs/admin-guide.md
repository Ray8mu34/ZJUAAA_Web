# 后台使用指南

本文面向日常维护网站内容的同学。正常更新网站内容时，不需要修改源代码，登录后台操作即可。

## 1. 后台入口

- 后台地址：`/admin`
- 登录地址：`/admin/login`
- 初始管理员账号只建议用于首次进入后台，进入后应尽快修改密码或新建个人管理员账号。

## 2. 日常维护原则

- 先上传图片，再创建或编辑内容。
- 知识科普和社团活动通常是“卡片入口 + 外部链接”模式。
- 天文手册是站内长期正文库，适合保存完整 Markdown 内容。
- 内部资料只给拥有内部账号密码的人访问，适合放社团内部文件和宣传部作品。
- 不确定图片路径时，优先使用媒体库选择，不要手动填写路径。
- 发布前检查状态是否为“已发布”。

## 3. 后台模块概览

| 模块 | 用途 |
| --- | --- |
| 首页管理 | 维护首页标题、副标题、按钮、Logo、首页展示内容等 |
| 站点设置 | 维护关于我们、加入我们、联系我们、历届成员、手册入口等 |
| 知识科普 | 维护科普文章卡片和公众号链接 |
| 社团活动 | 维护活动卡片、时间、地点、封面和跳转链接 |
| 天文手册 | 维护站内手册文章 |
| 手册栏目管理 | 创建和维护手册栏目 |
| 手册批量导入 | 用 ZIP 一次导入一个栏目的文章和图片 |
| 天文摄影 | 维护摄影作品和参数 |
| 媒体库 | 上传和选择图片 |
| 内部资料 | 管理内部下载文件和宣传部作品 |
| 管理员 | 新增管理员、停用账号、重置密码 |

## 4. 媒体库与图片

推荐流程：

1. 进入后台媒体库。
2. 上传图片。
3. 回到对应内容模块。
4. 在封面图、Logo、正文插图或作品图位置选择刚上传的图片。
5. 保存内容。

图片建议：

- 知识科普和社团活动封面建议接近 `4:3`。
- 手册栏目封面建议使用横图。
- 手册文章封图可选；不填则前台不显示封图占位。
- 天文摄影作品应尽量上传清晰原图，并填写作者和器材信息。
- 宣传部作品可以是 A4、横图、长图或方图；前台时间线和展开视图会按原图比例显示。

## 5. 知识科普

常用字段：

| 字段 | 是否必填 | 说明 |
| --- | --- | --- |
| 标题 | 必填 | 卡片标题 |
| 作者 | 必填 | 作者姓名 |
| 摘要 | 可选 | 卡片简介，前台会自动截断 |
| 封面图 | 可选 | 从媒体库选择 |
| 公众号文章链接 | 可选 | 填完整链接，例如 `https://mp.weixin.qq.com/...` |
| 是否精选 | 可选 | 精选后可出现在首页推荐 |
| 状态 | 必填 | 只有已发布内容会在前台显示 |

前台展示方式：

- 卡片显示标题、摘要、作者和封面。
- 图片仅展示。
- 图片下方文字区域可点击跳转。

## 6. 社团活动

常用字段：

| 字段 | 是否必填 | 说明 |
| --- | --- | --- |
| 标题 | 必填 | 活动标题 |
| 摘要 | 可选 | 活动简介，前台会自动截断 |
| 封面图 | 可选 | 从媒体库选择 |
| 地点 | 可选 | 活动地点 |
| 开始时间 | 可选 | 活动开始时间 |
| 结束时间 | 可选 | 活动结束时间 |
| 外部链接 | 可选 | 公众号推文、报名页或活动详情页 |
| 状态 | 必填 | 只有已发布内容会在前台显示 |

## 7. 天文手册

天文手册用于保存长期资料，正文存储在网站数据库中。

栏目管理可以：

- 新增栏目。
- 修改栏目标题、简介、封面和排序。
- 控制栏目是否对外可见。
- 删除栏目。

文章管理可以：

- 新增文章。
- 按栏目筛选文章。
- 编辑 Markdown 正文。
- 插入图片和公式。
- 设置文章封图；不填则不显示封图。
- 发布或保存草稿。

批量导入适合一次导入一个栏目的 Markdown 文件和图片。frontmatter 中的 `title`、`slug`、`author`、`summary` 等会作为元数据读取，不会显示进正文。

## 8. 内部资料

内部资料前台入口是 `/internal`，访问者需要输入内部资料账号和密码。这个账号密码由服务器环境变量配置，不是后台管理员账号。

后台入口：

- `/admin/internal`
- `/admin/internal/files`
- `/admin/internal/publicity`

### 8.1 内部文件

适合上传：

- 望远镜操作视频
- 培训资料
- 图文说明
- 社团内部文件

常用字段：

| 字段 | 是否必填 | 说明 |
| --- | --- | --- |
| 标题 | 必填 | 前台展示名称 |
| 描述 | 可选 | 文件说明 |
| 分类 | 可选 | 例如“望远镜”“培训”“宣传素材” |
| 排序值 | 可选 | 数字越小越靠前 |
| 文件 | 新增必填 | 上传实际文件 |
| 状态 | 必填 | 已发布才会出现在 `/internal/files` |

更新文件时，如果不重新上传文件，则只更新标题、描述、分类、排序和状态。

### 8.2 宣传部作品

宣传部作品前台入口是 `/internal/publicity`，包含顶部 3D 作品墙和下方作品时间线。

常用字段：

| 字段 | 是否必填 | 说明 |
| --- | --- | --- |
| 标题 | 必填 | 作品标题 |
| 图片 | 必填 | 从媒体库选择，媒体分类建议选“内部资料” |
| 作者 | 可选 | 不填时默认“天小协” |
| 日期 | 可选 | 前台按时间展示 |
| 描述 | 可选 | 点击作品和时间线中展示 |
| 排序值 | 可选 | 保留给手动排序 |
| 状态 | 必填 | 已发布才会出现在前台 |

展示建议：

- 作品图片比例不限。
- 顶部 3D 作品墙会统一成固定缩略比例，以保持视觉效果。
- 点击展开和时间线中的大图会按原图比例显示，避免裁切或拉伸。
- 描述较长时，前台会在玻璃信息卡内部滚动。

## 9. 天文摄影

常用字段：

- 标题
- 摄影者
- 作品图片
- 作品说明
- 目标天区
- 拍摄地点
- 主镜、相机、赤道仪、滤镜、软件
- 状态

只有已发布作品会在前台显示。

## 10. 关于我们与历届成员

关于我们页面包括：

- 社团介绍
- 照片墙
- 历届成员名单

历届成员支持逐条添加或批量导入。批量导入示例：

```text
2024,张三,会长
2024,李四,宣传部部长
2023,王五,学术部部长
```

前台会按年份倒序显示，最新年份在前。

## 11. 管理员账号

管理员模块可用于：

- 新增管理员。
- 停用账号。
- 重置密码。

建议：

- 每位维护者使用自己的账号。
- 不要多人共用初始管理员账号。
- 密码重置后应提醒对方尽快修改。

## 12. 不需要改源代码的内容

以下内容都应通过后台维护：

- 首页文案、按钮、Logo
- 页脚信息
- 关于我们、加入我们、联系我们文案
- 历届成员名单
- 知识科普卡片
- 社团活动卡片
- 天文手册栏目和文章
- 天文摄影作品
- 内部文件
- 宣传部作品
- 管理员账号

需要新增页面、大改页面布局、接入新服务或修改数据库结构时，再联系开发维护者。
