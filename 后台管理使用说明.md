# 后台管理使用说明

这份说明面向日常维护同学。正常更新网站内容时，不需要改源码，直接登录后台操作即可。

## 1. 后台入口

- 后台地址：`/admin`
- 登录地址：`/admin/login`
- 初始管理员账号请只用于第一次进入，登录后建议马上改密码

## 2. 各栏目工作方式

| 栏目 | 说明 |
|------|------|
| 首页 | 站内展示，文案、按钮、主视觉图、logo 等从后台读取 |
| 知识科普 | 卡片展示，点击跳转公众号文章 |
| 社团活动 | 卡片展示，点击跳转公众号或外部链接 |
| 天文手册 | 分栏目管理，正文保存在网站数据库里 |
| 天文摄影 | 站内展示作品信息 |
| 关于我们 / 加入我们 / 联系我们 | 站内展示，从后台读取 |

## 3. 后台模块说明

### 首页管理

维护首页标题、按钮、主视觉图片、logo、页脚基础信息。

### 站点设置

维护关于我们、加入我们、联系我们等页面的文案，以及历届成员名单。

**历届成员管理：**
- 逐条添加：点击"新增年份"和"新增成员"，手动填写姓名、职务
- 批量导入：点击"批量导入"，粘贴文本内容，每行格式 `年份,姓名,职务`，自动解析导入
- 支持的分隔符：逗号、制表符、顿号、中文逗号

批量导入示例：
```
2024,张三,会长
2024,李四,宣传部部长
2023,王五,学术部部长
```

**天文手册顶部内容：**
- 在站点设置中找到"天文手册顶部内容"文本框
- 粘贴 Markdown 格式的内容清单，保存后在 `/manual/start` 页面渲染
- 手册首页的"第一次来到这里？"入口会跳转到该页面
- 留空则显示默认提示

### 知识科普

维护科普卡片，不在网站内写长文正文，而是填写公众号文章链接。

### 社团活动

维护活动卡片，填写时间、地点、封面和跳转链接。

### 天文手册

分栏目管理的站内内容系统。包括：

- **栏目管理**：创建和管理手册栏目（如"观测手册"、"天文学入门"等）
- **文章管理**：在栏目下创建和编辑文章
- **批量导入**：通过 ZIP 文件一次性导入整个栏目的文章

### 天文摄影

维护摄影作品标题、说明、封面和详情。

### 媒体库

上传图片，供首页、科普、活动、摄影、手册等模块调用。

### 管理员

新增管理员、停用账号、重置密码。

## 4. 图片上传流程

1. 先进入 `媒体库`
2. 上传图片
3. 再进入对应内容模块
4. 在封面图、主视觉图或 logo 处选择刚上传的图片
5. 保存

后台大多数地方已支持直接从媒体库选图，不需要手敲路径。

**编辑器内快捷方式**：手册编辑器支持直接拖拽图片或粘贴截图，图片会自动上传到媒体库并插入到正文中。

## 5. 知识科普

科普文章是"展示卡片 + 跳公众号"模式。

| 字段 | 是否必填 | 说明 |
|------|---------|------|
| 标题 | 必填 | 文章标题 |
| 作者 | 必填 | 作者名 |
| 封面图片 | 可选 | 从媒体库选择 |
| 摘要 | 可选 | 卡片上显示的简介 |
| 公众号文章链接 | 可选 | 填完整链接，如 `https://mp.weixin.qq.com/...` |
| Slug | **不填** | 系统自动生成，无需理会 |

发布后，前台会显示卡片；点击按钮后，用户跳转到公众号原文。

## 6. 社团活动

社团活动是"展示卡片 + 跳外部链接"模式。

| 字段 | 是否必填 | 说明 |
|------|---------|------|
| 标题 | 必填 | 活动标题 |
| 摘要 | 可选 | 卡片上显示的简介 |
| 封面图片 | 可选 | 从媒体库选择 |
| 地点 | 可选 | 活动地点 |
| 开始时间 | 可选 | 活动开始时间 |
| 结束时间 | 可选 | 活动结束时间 |
| 外部链接 | 可选 | 公众号文章或活动页面链接 |

## 7. 天文手册

天文手册是站内长期保存内容，采用分栏目结构。

### 7.1 栏目管理

进入后台 → 栏目管理，可以：

- 新增栏目（设置标题、简介、封面图、排序）
- 编辑栏目信息
- 切换栏目是否对外可见
- 删除栏目（需先清空栏目下的文章）

默认栏目：观测手册、天文学入门、物理机制、天文学史、问题导向专题、知识卡片库。

**栏目字段说明：**

| 字段 | 是否必填 | 说明 |
|------|---------|------|
| Slug | 不填 | 系统自动生成 |
| 栏目标题 | 必填 | 如"观测手册" |
| 排序值 | 可选 | 数字越小越靠前，默认追加到最后 |
| 栏目简介 | 可选 | 栏目卡片上显示 |
| 栏目封面图 | 可选 | 从媒体库选择，16:9 效果最佳 |
| 对外可见 | 默认勾选 | 取消勾选则前台不展示 |

### 7.2 文章管理

进入后台 → 天文手册，可以：

- 新增文章（必须选择所属栏目）
- 按栏目筛选文章
- 编辑文章内容（支持 Markdown + 公式）
- 发布 / 设为草稿

**文章字段说明：**

| 字段 | 是否必填 | 说明 |
|------|---------|------|
| 所属栏目 | 必填 | 下拉选择 |
| 章节标题 | 必填 | 文章标题 |
| 排序值 | 可选 | 数字越小越靠前，留空则自动排到最后 |
| 作者 | 可选 | |
| 简介 | 可选 | 用于目录页展示 |
| Markdown 正文 | 必填 | 支持拖拽/粘贴图片 |

**不需要填写的字段：**
- Slug：系统从标题自动生成
- 章节编号：系统根据排序值自动生成
- 封面图：文章不需要封面图

**编辑器功能：**
- 支持 Markdown 语法
- 支持数学公式（LaTeX）
- 支持拖拽图片上传（自动上传到媒体库）
- 支持粘贴截图上传
- 支持从媒体库选择已有图片插入

### 7.3 批量导入

进入后台 → 批量导入，可以：

- 上传 ZIP 文件，一次性导入整个栏目的文章和图片
- 自动创建栏目（根据文件夹名称）
- 自动上传图片到媒体库
- 自动替换 Markdown 中的图片路径
- 自动读取 YAML frontmatter 中的 title、slug、author 等字段

**ZIP 文件结构要求：**

```
栏目名称/
├── 01-文章标题.md
├── 02-另一篇文章.md
└── images/
    ├── 图片1.jpg
    └── 图片2.png
```

**Markdown frontmatter 示例：**

```yaml
---
title: "为什么行星会逆行？"
slug: why-retrograde
author: 张三
summary: 文章简介
---
```

批量导入时的字段处理：

| 字段 | 导入行为 |
|------|---------|
| title | 优先使用 YAML 中的 title，否则从文件名去掉数字前缀 |
| slug | 优先使用 YAML 中的 slug，否则从标题自动生成 |
| author | 读取 YAML 中的 author |
| summary | 读取 YAML 中的 summary |
| 排序值 | 从文件名前缀提取（如 `01-xxx.md` → 1），否则自动递增 |
| 章节编号 | 自动等于排序值 |

## 8. 不需要改源码的内容

以下内容都应在后台修改：

- 首页主标题、副标题、按钮文案
- 首页图片和 logo
- 页脚联系方式
- 关于我们 / 加入我们 / 联系我们文案
- 历届成员名单
- 科普卡片
- 活动卡片
- 天文手册栏目和文章
- 摄影作品
- 管理员账号

## 9. 需要找开发者的情况

- 想新增一个全新页面
- 想大改页面布局和动画
- 想接入真正的多语言翻译系统
- 想接入站内表单提交、邮件通知、对象存储等新功能
- 想调整数据库结构

---

## 10. 日常维护建议

- 发文章前：先传图，再建内容
- 更新科普：只录入卡片信息和公众号链接
- 更新活动：录入时间、地点、封面和跳转链接
- 更新手册：在网站内保留完整 Markdown 正文
- 每次新增管理员后，提醒对方尽快修改初始密码

---

## 11. 一句话理解网站结构

- **科普文章**：网站做目录和入口，正文在公众号
- **社团活动**：网站做目录和入口，详情在公众号或外部页面
- **天文手册**：网站自己保存正文，分栏目管理，作为长期内容和备份
